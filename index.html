About Me

Saya Nurjannah sulaimatul firda, biasanya teman-teman saya memanggil saya dengan nama firda, parida, fir,par dsb. Terdengar aneh tapi saya menyukainya. Menurut saya mungkin itu suatu panggilan 'sayang' dari mereka untuk saya. Saya adalah anak kedua dari dua bersaudara, ya benar saya anak bontot.

Sebelum memasuki SMKN 1 Tanjungpinang, saya bersekolah di SMPN 1 Tanjungpinang. Meskipun saya hanya sekolah satu tahun (sisanya covid-19) membuat masa-masa putih biru saya tidak memiliki kenangan yang berarti, tetapi tidak menutup kemungkinan pula saya merindukan masa-masa itu.

Saya dari jurusan manajemen perkantoran, terdengar aneh jika anak perkantoran yang terbiasa mengetik surat kini beralih mengetik coding. Awalnya memang saya takut untuk menjalani ini, tetapi seiring berjalannya waktu saya mulai menerimanya. "Ternyata tidak terlalu buruk kok" ucap saya. Meskipun saya masih pemula dan baru mempelajari HTML dan CSS saya harap kedepannya saya bisa terus mempelajari ilmu-ilmu baru.
